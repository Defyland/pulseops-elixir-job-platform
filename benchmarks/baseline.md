# Benchmark Baseline

This folder contains the benchmark scripts used to measure the PulseOps API.

## Scenarios

- `smoke.js`
- `load.js`
- `stress.js`
- `spike.js`

## Environment assumptions

- API available at `http://localhost:4000`
- `PULSEOPS_API_KEY` exported
- at least one tenant with the `default` queue provisioned
